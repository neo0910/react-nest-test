## Preparations

Docker, yarn

## Manual

```bash
$ docker-compose up -d
```

### Server

```bash
# from root dir
$ cd server
$ yarn install
$ yarn start:seed

$ yarn start
```

## Client

```bash
# from root dir
$ cd client
$ yarn install

$ yarn start
```

## Links

- client - [http://localhost:5173](http://localhost:5173/)
- server - [http://localhost:3000](http://localhost:3000/)
